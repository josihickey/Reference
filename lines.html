Basic Code

To create a line, use the < hr > tag. There is no end tag for lines. This will create a horizontal line. This line will be largely transparent, with the exception of a border. Here is a basic line:




Style

There are many ways to style a line, but they never seem to consistently work. I included a few tags that have at some point worked for me. Good Luck!

This image created the purple line, and the odd line that I don't totally know what to do with next to it. I've found the "background-color" function to be the most helpful with changing the line's thickness. The height alters the height (obviously) or thickness of the line. Border width alters the border, or non-transparent aspect of the original line. For some reason, using "pc" made it very thick. I haven't figured out why yet. Either way, the second line (#weirdline) shows the use of "border-width". Using "color" however, as I did in the second code, does absolutely nothing.

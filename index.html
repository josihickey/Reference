Tables

Command Function
table Creates a table (the basic command)
thead Groups the heading cells
tbody Groups the body cells
tr Creates a row
th Adds a column in a row in the heading. Makes it bold (and centered?) Goes inbetween "< tr >" and "< /tr >"
td Same as th, but in the body.
